How to Play Number Harmony

Welcome to Number Harmony, a magical adventure where you'll match numbers with their equivalent expressions to create beautiful harmonies! This guide will help you understand how to play the game and have fun while learning math.

What is Number Harmony?

Number Harmony is a memory matching game where you need to find pairs of cards that represent the same number. One card shows the number itself, while the matching card shows a math expression that equals that number.

Example:

The card showing "8" matches with the card showing "4+4" because 4+4 equals 8.

8
4+4

How to Play:

  • When you start the game, you'll see a grid of cards face down.
  • Click on any card to flip it over and see what's on the other side.
  • Then click on another card to see if it matches the first one.
  • If the two cards match (represent the same number), they will stay face up.
  • If they don't match, they will flip back face down, and you'll need to try again.
  • Your goal is to find all the matching pairs with the fewest moves and in the shortest time.

Magical Tips:

  • Try to remember where each card is after it's flipped back over.
  • Look for patterns in the expressions. For example, cards with addition (like 3+3) or multiplication (like 2×2).
  • Start by finding the matches for the simplest numbers first.
  • Take your time - it's not just about speed, but also about understanding the math relationships!

Math Skills You'll Practice:

  • Addition (recognizing that 3+3 equals 6)
  • Multiplication (recognizing that 2×2 equals 4)
  • Number equivalence (understanding different ways to represent the same number)
  • Memory skills (remembering where cards are located)

Controls:

  • Click on a card to flip it over
  • Start Adventure button begins a new game with freshly shuffled cards
  • Reset button lets you start over if you want to try again

When you find all the matching pairs, you'll be rewarded with a magical celebration! The game will show you how many moves you made and how much time it took you to complete the adventure.
