Number Strike Guide

Learn how to complete the Number Strike mission and defeat the math monsters!

Mission Briefing: Number Strike

Welcome to Number Strike, an action-packed mission where you'll match numbers with their equivalent expressions to defeat the math monsters! This guide will prepare you for your mission and help you succeed.

Mission Objective

Number Strike is a memory matching game where you need to find pairs of cards that represent the same number. One card shows the number itself, while the matching card shows a math expression that equals that number.

Target Example:

The card showing "8" matches with the card showing "4+4" because 4+4 equals 8.

8
4+4

Mission Protocol:

  • When you start the mission, you'll see a grid of cards face down.
  • Click on any card to flip it over and reveal what's on the other side.
  • Then click on another card to see if it matches the first one.
  • If the two cards match (represent the same number), they will stay face up.
  • If they don't match, they will flip back face down, and you'll need to try again.
  • Your objective is to find all the matching pairs with the fewest moves and in the shortest time.

Tactical Tips:

  • Memorize the location of each card after it's flipped back over.
  • Look for patterns in the expressions. For example, cards with addition (like 3+3) or multiplication (like 2×2).
  • Start by finding the matches for the simplest numbers first.
  • Develop a strategy - work systematically through the grid rather than clicking randomly.

Skills You'll Master:

  • Addition (recognizing that 3+3 equals 6)
  • Multiplication (recognizing that 2×2 equals 4)
  • Number equivalence (understanding different ways to represent the same number)
  • Strategic thinking and memory skills

Mission Controls:

  • Click on a card to flip it over
  • Start Mission button begins a new mission with freshly shuffled cards
  • Abort Mission button lets you restart if you want to try again

When you find all the matching pairs, you'll complete the mission! The system will show you how many moves you made and how much time it took you to defeat the math monsters.
